## Docker Command
### โครงสร้างคำสั่ง
    docker <command>

### docker --version, docker version
    ตรวจสอบเวอร์ชันของ docker

### docker info
    แสดงรายละเอียดของ docker ทั้งหมด

### docker system df
    แสดงข้อมูลการใช้พื้นที่ disk ของ docer

### docker --help
    ดูคำสั่ง docker ทั้งหมด

### docker images
    แสดง images ที่มีอยู่ทั้งหมดบนเครื่องเรา
